# Линейная регрессия - прогноз прибыли автосервиса

Прогнозирование прибыли станции техобслуживания автомобилей на основе количества автомобилей в населённом пункте с использованием линейной регрессии.

## Задача
Разработать систему искусственного интеллекта, прогнозирующую прибыль СТО в зависимости от количества автомобилей в населённом пункте.

## Данные
Файл `ex1data1.txt`:
- Первый столбец: количество автомобилей (×10,000)
- Второй столбец: прибыль СТО (×10,000$)

## Математическая модель

### Линейная регрессия
hθ(x) = θ0 + θ1x
где:
- hθ(x) - предсказанное значение
- θ0, θ1 - параметры модели  
- x - количество автомобилей (в 10,000s)

### Функция стоимости
J(θ) = 1/2m * Σ(hθ(x(i)) - y(i))^2
где m - размер обучающей выборки

### Градиентный спуск
θj = θj - α * ∂/∂θj J(θ)

Для линейной регрессии:
θ0 = θ0 - α * 1/m * Σ(hθ(x(i)) - y(i))
θ1 = θ1 - α * 1/m * Σ(hθ(x(i)) - y(i)) * x(i)

## Запуск

1. Установи зависимости:
```bash
pip install -r requirements.txt
```

2. Запусти обучение модели:
```bash
python src/ex1.py
```

3. Для интерактивного прогнозирования:
```bash
python src/work.py
```

4. Запусти тесты:
```bash
python src/test_cost.py
```
